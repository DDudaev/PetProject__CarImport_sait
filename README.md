# üßæ Project Description

CarImport is a fully featured pet project created to demonstrate proficiency with a modern development stack:

- Nuxt 4.2.1

- TypeScript

- Pinia Store

All global styles, colors, variables, and resets are located in /assets/css/.

The website is fully responsive (desktop + tablet + mobile).
